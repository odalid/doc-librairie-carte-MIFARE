ODALID Librairie Carte MIFARE

Introduction

Cette documentation présente les différentes fonctions et leurs paramètres contenus dans la librarie.

La librairie est compatible avec les environnements windows et linux.

Cette librairie est compiler avec le compilateur open source de MinGW 8.1.0 64-bit

Presentation

La librairie ODALID est composé de 4 parties :

  • Plugin;
  • Core;
  • Extension;
  • Application.
